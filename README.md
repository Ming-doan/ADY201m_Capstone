# ADY201m Capstone project Front-end

This is Front-end part of ADY201m Capstone Project

## Technologies
- React
- React Router
- Zustand
- Ant Design
- React Icons
- Axios

## How to run
Clone this project
```
git clone <this_repo_url>
```

Install dependencies
```
npm
```

Run project locally
```
npm start
```
